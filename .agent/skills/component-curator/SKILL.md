# Component Curator Skill

Before building any component:

1. Search for inspiration from:

* shadcn/ui
* Magic UI
* Aceternity UI
* 21st.dev

2. Evaluate component quality.

3. Improve component before implementation.

Never copy components blindly.

Judge every component by:

* visual quality
* reusability
* accessibility
* responsiveness
* animation potential

Preferred component style:

* glassmorphism
* luxury minimalism
* premium shadows
* subtle glow
